# twine-test

## prerequisites
- git
- node + npm

## getting started
- clone the repo
- install node modules with `npm install`
- add `.twee` file(s) to `src/stories/`

## staging local build
- run extwee compiler build with `npm run build`
- run local server with `npm run serve`

## push changes up to github & deploy result to github pages
- `git add .`
- `git commit -m "MY MESSAGE"`
- `git push`

> Note: you can run `git status` between each step to check in.
